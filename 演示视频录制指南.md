# 演示视频录制指南

## 推荐工具

| 工具 | 平台 | 说明 |
|------|------|------|
| OBS Studio | 免费/Win/Mac/Linux | 专业录屏，支持区域录制 |
| Xbox Game Bar | Win 10/11 自带 | 按 `Win+G` 调出，简单易用 |
| Screen Studio | macOS 付费 | 高质量录屏 + 自动剪辑 |

## 视频要求

- **时长**：5 分钟以内
- **格式**：MP4（H.264 编码）
- **内容**：语音解说 + 屏幕操作演示
- **字幕**：建议添加（可用剪映/必剪自动生成后校对）

## 演示脚本（约 4 分钟）

### 开场 (0:00-0:20)
> "大家好，今天演示的是情感良师 LoveMentor，一个面向爱情与两性关系的 NLP 情感分析系统。我叫它白开水——一个温暖的情感导师。"

### 功能 1：情感倾诉 (0:20-1:40)
1. 展示首页欢迎界面（白开水介绍）
2. 输入一段情感文本：「TA 今天一整天都没回我消息，我是不是想太多了...」
3. 点击发送，等待 AI 回应
4. **重点展示**：情感标签（焦虑/负面）、温暖回应内容
5. 再输入一段正面文本：「今天和 TA 一起看了日落，很幸福」
6. 展示情感标签变化（幸福/正面）+ 不同风格的回应

### 功能 2：星座分析 (1:40-3:00)
1. 切换到"星座分析"Tab
2. 演示星座匹配：选择「天蝎座」+「双鱼座」
3. **重点展示**：95 分匹配度、优势、挑战、浪漫建议
4. 演示今日运势：选择「天蝎座」
5. 展示运势卡片：爱情指数、幸运色、今日小贴士

### 功能 3：心情记录 (3:00-3:30)
1. 切换到"心情记录"Tab
2. 展示之前的对话记录列表
3. 说明时间线展示方式

### 总结 (3:30-4:00)
> "以上就是情感良师的核心功能。系统基于 Flask + DeepSeek API 构建，支持降级运行，无 API Key 也能正常演示。谢谢观看！"

## 后期处理建议

1. 使用**剪映**或**必剪**自动生成字幕，校对修正专业术语
2. 适当 1.2x-1.5x 加速无操作等待片段
3. 在关键操作处添加文字标注（如"情感标签：焦虑"）
4. 音量调整至适中，背景噪音降噪

## 注意事项

- 提前配置好 DeepSeek API Key，确保演示时 API 调用正常
- 关闭无关浏览器标签页和通知弹窗
- 录屏区域选择浏览器窗口（建议 1920×1080）
- 准备耳机麦克风，确保语音清晰
- 提前演练一遍，控制好节奏
